# Smart Folder Organizer

A Python automation project that automatically organizes files into folders based on their file extensions.

## Features

- Organizes Images, PDFs, Videos, Music, Text, and Python files
- Creates folders automatically if they don't exist
- Uses Python's `os` and `shutil` modules
- Fast and easy to use

## Technologies Used

- Python 3
- os
- shutil

## How to Run

```bash
python organizer.py
```

## Example

Before:

photo.jpg
movie.mp4
resume.pdf

After:

Images/
    photo.jpg

Videos/
    movie.mp4

PDFs/
    resume.pdf
